Questions:

1. What are 4 ways HTML is used? - Contain content, control how it is displayed, how search engines perceive a web page and what scripts it can run. 

2. What are HTML files made up of? - Tags and content

3. What are the two formats for most tags and how are they displayed? - An opening/closing tag has a tag before the affected content and a closing or end tag after. A self-closing tag only has a single tag, All self-closing tags in html 5 start with a less than sign followed by the tag name, and end with "space, forward slash, greater than"

4. What tags should all HTML files have? - document type declaration, body tag, head tag, html tag and title tag. 

5. What tag is used to create hyperlinks and what is it's notation?? - Anchor tag 

6. What will happen to your webpage if you do not inclue a unique 'meta description'? - It will cause search engines to provide either no or a wrong description on results page

7. Which tags can only appear once on a web page? - document type declaration, body tag, head tag, html tag and title tag

8. How to HTML files end, how is white space displayed and how do you view the source code of a web page? - Ends with .html or .htm, all white space is displayed as a single space unless specified otherwise using tags, Ctrl-U to show a web page's source code
