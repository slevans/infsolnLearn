Questions:

1. What two things make up a domain name? - unique name and a domain suffix

2. What is the difference between a relative and absolute path? - relative path lists the directories relative to the current location in the file system, while an absolute path lists the directories from the base of the files system.

3. What is the specific location for a file on a network? - Uniform Resource Identifier (URI)

4. This kind of error indicates the correct server was found but the page was not. - 404

5. Go Daddy is an example of what? - Registrar

6. What do you call a private network that can only be accessed by the company who created it? - intranet

7. What is the portion of the internet that handles http traffic? - World Wide Web

8. Where should you never store your SSN, User ID or other private information? - a cookie
