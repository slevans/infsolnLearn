1. What does the gateway at the hosting facility for infsoln.com do once it receives the request for the web page? - it sends it on to the infsoln.com server.

2. When a file needs to get data from the database what does php use to send a request to the database? - the SQL query found in the code

3. What does the web server do once php completes parsing the code and turns it into html? - It sends the generated html code back to the gateway

4. What acts as 'the client' in this process? - your computer

5. What role does the ISP play? - it passes the request to the internet, routes the http request to the server at the IP Address given by the Domain Name Registry and in the end returns the html from the server to the computer

6. What does the web server do when no specific file is requested? - it grabs the first available default.

7. How, in the end of the process, is the web page displayed? - Your browser receives the html code, parses it and that displays what you end up seeing on the page.

8. What is DNS and what is its role in the process? - Domain name service, this is how a computer looks up where to find the domain name it is looking for.
